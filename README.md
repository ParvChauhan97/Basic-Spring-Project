# 🚀 Basic Spring Boot Project
This project is part of my journey to learn and build backend applications using Spring Boot.

## Project Goal
The main goal of this project is to understand how to build REST APIs and implement CRUD (Create, Read, Update, Delete) operations step by step.

---

## What I Will Build?
In this project, I will be implementing:

-> Create User API  
-> Get All Users API  
-> Get User by ID  
-> Update User API  
-> Delete User API  

---

## What I Will Learn?
Through this project, I aim to learn:

--> Basics of Spring Boot  
-->  REST API Development  
-->  Understanding Controller, Service layers  
-->  Handling HTTP Requests (GET, POST, PUT, DELETE)  
-->  Project structure and clean coding practices  

---

## Note
This is a beginner-friendly project focused on learning and improving backend development skills.
